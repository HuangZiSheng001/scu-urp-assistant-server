# 四川大学综合教务系统助手后端

## Description

四川大学综合教务系统助手后端，基于 [Nest.js](https://github.com/nestjs/nest) 框架开发。

## 安装

```bash
$ npm install
```

## 运行

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
